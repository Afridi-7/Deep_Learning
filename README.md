# Deep Learning Experiments (PyTorch)

This repository contains a collection of **deep learning experiments and implementations** created while studying neural networks and modern deep learning techniques.

The goal of this repository is to **understand deep learning concepts by implementing models and training pipelines from scratch**, mainly using **PyTorch**.

New projects and experiments will continue to be added as the learning process progresses.

---

## 📚 Topics Covered

The repository includes experiments related to:

* Neural networks from scratch
* Perceptron implementation
* PyTorch tensors and autograd
* Training pipelines using `torch.nn`
* Artificial Neural Networks (ANN)
* Convolutional Neural Networks (CNN)
* Transfer learning
* Regularization and overfitting analysis
* Object detection using **YOLOv8**

---

## 📂 Current Projects

| Notebook                                          | Description                                                    |

| `tensors.ipynb`                                   | Introduction to PyTorch tensors and tensor operations          |

| `Transfer_Learning_fashion_mnist_pytorch.ipynb`   | Applying transfer learning to improve performance              |
| `blood_cells_detection_with_yolov8.ipynb`         | Blood cell detection experiment using YOLOv8                   |
